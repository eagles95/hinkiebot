NHL Commands(work in progress tbh fam)

All nhl commands start with "#"

Command Description
#score teamName Returns the score of that teamName latest game
#stats playerName Returns common stats for that playerName season
#livestats playerName Returns common stats for that playerName latest game
#commands or #info Returns this page

Arguments

Argument Name Options
teamName Team name or nickname(Flyers, Avs, Avalanche, Habs, Canadiens, etc)
playerName Could be first name and last name (Claude Giroux,Wayne Simmonds, etc) or just one name (Giroux, Wayne, etc)/ if there are multiple players with that name it just returns whovever it finds first.

NBA Commands

All nba commands start with "/" or "&"

Command Description
/score teamName Returns the score and period of that teamName latest game
/nextgame teamName Returns opponent,start time and date of the next game for that teamName
/last5 teamName Returns scores of that teamName last 5 games
/record teamName Returns record and placement in conference for that teamName
/teamstats teamName Returns common stats for that teamName latest game
/standings conference/division name/tank Returns full standings for that conference/division
/scoreboard Returns scores for today's games
/recent playerName Returns stats for that playerName last 3 games
(/seasonstats playerName Returns playerName season averages
(/stats or /s) playerName Returns stats for that playerName current/latest game
/tdalert playerName Returns a triple double alert for that playerName
/profile playerName Returns picture,team,number,height,weight,age for that playerName
/quote Returns one of Sam Hinkie's idioms/quotes, full list here
/commands or /info Returns this page

Arguments

Argument Name Options
teamName Could be the city name(portland, toronto, philadelphia, etc),team name(mavericks, thunder, etc), both city and team name(houston rockets, utah jazz, etc), the abbreviation(PHI, LAL, BOS, etc) or a nickname (cavs, mavs, raps, sixers, etc)
conference/division name/tank west, east, southwest, pacific, northwest, southeast, atlantic, central or tank (standings in reverse)
playerName Can be in format: firstName lastName(lebron james, joel embiid, etc). UPDATE : now accepts single name lookups, first or last names. If the name is not unique it just returns whoever it finds first (lebron,fultz,giannis,lonzo,etc)
